# Calculator
A modern dark-pink calculator built with HTML, CSS, and JavaScript.
This calculator supports basic arithmetic operations, real-time display updates, error handling, and a clean responsive interface.
##Technologies Used
HTML
CSS
JavaScript
##How to Use
1. Clone or download this project.
2. Open the project folder.
3. Double-click index.html to open it in your browser.
4. Use the on-screen buttons to perform calculations:
5. Numbers → enter values
   Operators → +, −, ×, ÷, %
   DEL → delete last digit
   CLR → clear all = → calculate result
6. No installation required — everything runs directly in the browser.
